# TATWO-ULTRAWORK beta v0.1.0-beta.1

Initial public beta.

## Included

- Public Goal-first TATWO Work OS constitution.
- Provider-neutral command and OpenAI-compatible HTTP adapters.
- stdio MCP server and cross-platform Node CLI.
- Goal, Context, Plan, Loop, alignment, and Goal Judge contracts.
- Restricted Docker sandbox with fail-closed fallback.
- Public `$tatwo-ultrawork-beta` Skill.
- Chinese and English documentation.
- Cross-platform tests and public-data safety scan.

## Not included

- Graphical TATWO OS App.
- Hosted gateway or account service.
- Device control, synchronization, or update infrastructure.
- Provider credential storage.
- Private project data or private runtime integrations.

This is a beta. Use synthetic data first and review every adapter and sandbox
policy before applying it to a sensitive project.
